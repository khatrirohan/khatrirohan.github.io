This Gatepass app does not need a printer, desktop computer, or any other infrastructure. This works seamlessly on Android smartphones. This application is extremely useful for cooperative housing societies, Office buildings, business centers, schools, and colleges. Wherever people are coming in and going out, this system will be useful to manage the traffic.

This application can be used for
1. Create Sales Leads.
2. Search existing customers.
3. Create new customer records.
4. Receive timely updates on leads.
5. Opportunity assessment.
6. Viewing relevant reports and dashboard.
